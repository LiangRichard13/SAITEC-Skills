# AIGC 检测 SOP

## 触发条件

当用户上传图片或文本，并询问是否为 AI 生成时，调用检测工具。

## 调用工具

- **图片检测**: `detect_image(image_url: str)`
- **文本检测**: `detect_text(text: str)`

## 结果解读

| 置信度 | 含义 |
|--------|------|
| >= 0.8 | 高置信度，AI 生成 |
| 0.5-0.8 | 中等置信度，需人工复核 |
| < 0.5 | 低置信度，人类创作 |

## 回复模板

检测完成后，根据结果按如下方式回复：

```
检测结果：{label}
置信度：{confidence:.2%}
{附加建议（可选）}
```
